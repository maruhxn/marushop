## Stack

- 프론트엔드: React.js + MUI
- 백엔드: Express.js + passport + prisma
- DB: PostgreSQL
- CI/CD: Docker + Github Actions

## Deploy

- 프론트엔드: vercel
- 백엔드: fly.io
- 스토리지: local storage
- DB: local postgresql
- Mail Service: AWS SES
