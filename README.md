## Stack

- 프론트엔드: Next.js + MUI
- 백엔드: Express.js + passport(session) + prisma
- DB: PostgreSQL
- CI/CD: Docker + Github Actions
- 형상 관리: Git + Github

## Deploy

- 프론트엔드: vercel
- 백엔드: AWS ECR + AWS App runner + AWS Lambda(+ AWS Event Bridge)
- 스토리지: S3
- DB: local postgresql
- Mail Service: AWS SES
